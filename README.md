# Pagina WEB Grupo Trono

Sitio web corporativo de Grupo Trono — portafolio, servicios y contacto.

## Stack

HTML + CSS + JS vanilla (sin framework).

## Estructura

```
.
├── index.html          Landing / home
├── nosotros.html       Sobre Grupo Trono
├── servicios/          Páginas de cada filial / servicio
├── portafolio.html     Portafolio de proyectos
├── videos.html         Reel / videos de producción
├── contacto.html       Formulario de contacto
├── blog/               Artículos
├── css/                Hojas de estilo
├── js/                 Scripts
└── img/                Assets gráficos
```

## Desarrollo local

Abrir `index.html` en navegador o servir con cualquier server estático:

```bash
python -m http.server 8080
# o
npx serve .
```

## Despliegue

Por definir (Cloudflare Pages / Vercel / Netlify / hosting propio).
